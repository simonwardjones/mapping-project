Mapping Project

Watching a film the other day, Indiana jones if you’re curious, I decided I wanted to make a map animation showing all the flights I had ever taken. Admittedly not my idea, just Inspired by the time compression map montage in the film.

I decided to use this as an excuse to get more familiar with the python coding language. First and by far the most tricky part was remembering all the flights I have taken! Once that was done and in a csv it was on to the coding. I decided I use, and would recommend using Jupyter notebook, especially when relatively new to python as it separates code and outputs serially making it easy to read and explore.

Eventually my mini project broke down into three sections. Look below for more detail but broadly it was:

  1. Getting the data into python (I used a pandas data frame)
  2. Creating the map image and flight paths ( this was made incredibly easy using the matplot lib package)
  3. Animating. ( Again this was a simple using the animation package)
  4. Using xxxx to make a pretty version for the web.
